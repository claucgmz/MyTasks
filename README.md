# My Tasks

Practice app for creating lists with tasks to be completed and managed.

## Branches

*   __master:__ Production-ready state
*   __develop:__ Development-ready state


## Built With

*   __Swift__
*   __Cocoapods__

### Cocoapods

*   __FacebookCore__
*   __FacebookLogin__
*   __RealmSwift__
*   __AlamofireImage__
*   __SlideMenuControllerSwift__
